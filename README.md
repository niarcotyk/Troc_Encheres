# Troc_Encheres
L'association "Les objets sont nos amis" désire monter une plateforme web pour permettre la cession d'objets de seconde main sans échanges financiers. 
La valeur des articles sera déterminée par un système d'enchères basée sur un nombre de points.
Les points sont gagnés en vendant des objets, puis peuvent être utilisés pour acquérir d'autres objets.
